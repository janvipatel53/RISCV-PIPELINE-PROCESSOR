# RV32I Processor in Verilog

A 32-bit RV32I processor implemented in Verilog.

## Features

- Program Counter
- Instruction Memory
- Register File
- ALU
- Control Unit
- Data Memory
- Single-Cycle Datapath

## Verified Instructions

- ADD
- ADDI
